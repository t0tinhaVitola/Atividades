Título do Problema: "Rotate List"

Descrição do Problema: "Dada a **head** de uma lista encadeada, rotaciona a lista para a direita **k** vezes."

Exemplos de entrada e saída do problema:

Entrada: 
  - head = {1, 2, 3, 4, 5};
  - k = 2;

Saída: {4, 5, 1, 2, 3};

Entrada: 
  - head = {0, 1, 2};
  - k = 4;

Saída: {2, 0, 1};

Entrada: 
  - head = {1, 2, 3};
  - k = 2000000000;

Saída: {2, 3, 1};

Complexidade do algoritmo final:

  - Tempo: O(n*k);
  - Espaço: O(1).
