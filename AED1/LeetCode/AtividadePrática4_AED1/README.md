Título do Problema: "Largest Number"

Descrição do Problema: "Dado um vetor de inteiros **nums**, retorna o maior número possível de ser adquirido através da combinação dos elementos do vetor. O resultado sendo uma string da maior combinação possível."

Exemplos de entrada e saída do problema:

Entrada: 
  - nums = {10, 2};

Saída: 210;

Entrada: 
  - nums = {90, 1, 40};

Saída: 90401;

Entrada: 
  - head = {3,30,34,5,9};

Saída: 9534330;

Complexidade do algoritmo final:

  - Tempo: O(n*log(n));
  - Espaço: O(n).


Detalhes da Implementação: 
  - Contexto: Atividade prática realizada na plataforma do LeetCode (Problema 179) Código-fonte: Disponível no arquivo ".c" nesta pasta.
